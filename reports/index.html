About QuireBind

FOSS desktop and web application for preparing PDF files for bookbinding

Project Modules

This project has declared the following modules:

Name Description
QuireBind Core PDF engine, imposition logic, and all business rules
QuireBind Desktop UI JavaFX desktop application
QuireBind Web UI Spring Boot REST API and HTMX/Thymeleaf web frontend
QuireBind Guides Bundled markdown binding guides, loader, and validator
QuireBind Batch CLI batch processor for unattended .quire file processing
QuireBind Test Test infrastructure, PDF generator, and system test suite
QuireBind Convert Format conversion pipeline — Phase 2 placeholder. No code in Phase 1.
